# lbd-react-memory-game
A simple memory game created with react.
[https://raglopa.github.io/lbd-react-memory-game/](https://raglopa.github.io/lbd-react-memory-game/)
### stack
- react
- redux
- sass
- webpack

### install
```
yarn
```

### dev
```
yarn run dev 
```

### build
```
yarn run build
```
